發想：
我們希望將繁瑣的登入步驟（輸入帳號密碼）等工作簡化成感應悠遊卡，如同圖書館的借書系統只需要悠遊卡即可快速借書。但為了保護學生證被拷貝或遺失的狀態，我們也提供了額外的保護。

基於我們作業二所建立的檔案權限系統我們將登入分為兩個層次：

  1. 開機時以學生證登入系統：只能使用 other 權限允許的內容，如： ------rwx
  2. 若要讀取私人權限的檔案或程式，則需要使用 gogopw(類似 sudo 的感覺) 另外輸入密碼。

+{
+    if (password[strlen(password) - 1] == '\n')
+        password[strlen(password) - 1] = '\0';
+
+    int fd;

+        for (int i = 0; i < n; ++i) {

結論

在這次的實作中，我們更深入理解了 xv6 的 file system 並將他完整改寫成類現代 Linux 的權限系統，同時也要兼顧安全性，所以在整個 kernel 讀檔寫檔的過程中都要處理權限問題。在將 RFID reader 串接的過程中，處理了較底層的 IO。

另外在實際結果上，能用 RFID 登入真的看起來蠻炫砲的，上面那個 gif 動圖超紓壓的（？），然後雙重登入權限也是一個蠻好玩的概念。
